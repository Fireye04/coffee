## Disco Tools

[Disco tools](https://codeberg.org/Fireye/disco-tools) has not only released, but has already recieved its first major update! 

Essentially, Disco Tools is a system that handles State and RNG to facilitate the creation of games in the vein of Disco Elysium. It builds off the splendid narrative base provided by Nathan Hoad's [Godot Dialogue Manager](https://github.com/nathanhoad/godot_dialogue_manager), to minimize the amount of work required to craft a narrative heavy game in godot.

This is the tooling from Gas Station, fully refactored into a solitary plugin from C# to GDScript and everything. I cannot tell you how much work this bitch has been, but I'm so glad it's out there.

## Codeberg Migration

You might notice that I've been migrating off github. 
Progress is slow, but it's going little by little. 

This is an ethical and political choice I'm making, as I dislike the proprietary nature of github, and the rapid enshittification therein, in addition to a general reluctance to use anything microslop-related due to the [boycott](https://bdsmovement.net/microsoft).
Moreover, Git federation is the way forward, and github will not be a proponent therein, so I would much rather support an alternative service that [is](https://codeberg.org/ForgeFed/ForgeFed).

Codeberg hasn't been flawless (It's down as I write this haha), but overall, it's been working just fine. 

Selfhosted forgejo has also been awesome, and I'd definitely recommend it. 
I've been running a generational dwarf fortress game off it with a friend, and it's been super fun and convenient to have a personal git server for stupid shit like that.
